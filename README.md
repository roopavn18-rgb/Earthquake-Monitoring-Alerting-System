# Earthquake Monitoring and Alerting System

# Description
This project is a low-cost earthquake detection system using Arduino Uno and SW-420 vibration sensor. It detects abnormal vibrations and provides instant alerts using buzzer, LED, and LCD display.

## Components Used
- Arduino Uno
- SW-420 Vibration Sensor
- 16x2 LCD Display
- Buzzer
- LED
- Resistors, Breadboard, Jumper Wires

# Working Principle
The vibration sensor detects ground movement and sends signals to Arduino. When vibration exceeds a threshold, the system activates buzzer and LED, and displays "Earthquake Alert" on LCD.

# Features
- Real-time vibration detection
- Instant alert system (LED + Buzzer)
- LCD status display
- Low-cost and easy to implement

## Applications
- Home safety systems
- Schools and labs
- Small buildings and offices
